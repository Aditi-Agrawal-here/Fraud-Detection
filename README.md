# Fraud-Detection
Project Overview : 
This project is a Machine Learning–based Fraud Detection System developed to identify fraudulent financial transactions. The model analyzes transaction details and predicts whether a transaction is fraudulent or legitimate. The system is deployed using Streamlit, providing an interactive and user-friendly web interface for real-time predictions. 
Features Machine Learning–based fraud prediction Interactive Streamlit web application Real-time transaction analysis Clean and simple user interface Trained model saved as a reusable pipeline Technologies Used Python Pandas Scikit-learn Streamlit Joblib
